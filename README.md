## Pre-install:
VSCode  
Python3

pip3 install pygame
## Intro
競賽系統: `main.py`  
競賽系統設定: `game_set.py`  
模板: `template.py`  
實用函式: `simplelib.py`  
常數: `variables.py`  
## Execute
請先於`game_set.py`進行設定後，執行`python3 main.py`。  
注意! 一定要裝python3，否則系統無法執行你的程式
